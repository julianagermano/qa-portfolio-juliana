
# Qualidade Orientada a Produto (QA Engineering)

Este conjunto de documentos define como a Qualidade é aplicada com foco em **valor de negócio**:
- priorização por jornadas do usuário (journeys)
- riscos por impacto/probabilidade
- critérios de aceite + critérios técnicos de qualidade
- regras objetivas de Go/No-Go
- métricas e SLOs por jornada

Objetivo: apoiar decisões de liberação com base em evidências e reduzir risco de produção.
