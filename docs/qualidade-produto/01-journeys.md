# 01 — Jornadas do Usuário (User Journeys)

## Journeys principais
1. Login / Autenticação
2. Criar solicitação / Pedido
3. Confirmar / Pagamento (se aplicável)
4. Consultar status
5. Cancelar / Reverter
6. Suporte / Contato

## Critérios de “fluxo crítico”
Uma jornada é crítica quando:
- gera receita ou evita perda financeira
- bloqueia uso do produto quando falha
- tem impacto reputacional alto
- tem alto volume de usuários
