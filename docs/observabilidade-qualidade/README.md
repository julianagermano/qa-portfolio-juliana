# Observabilidade aplicada à Qualidade

Este conjunto de documentos descreve como dados de produção
(métricas, logs e eventos) são utilizados como insumo para
estratégia de testes, quality gates e decisões de liberação.

Objetivo:
- reduzir incidentes
- orientar testes por risco real
- apoiar Go/No-Go com dados
- promover melhoria contínua da qualidade
