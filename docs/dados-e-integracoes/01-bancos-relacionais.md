# Bancos de Dados Relacionais — Visão de Qualidade

Bancos relacionais utilizam tabelas estruturadas, relacionamentos
e regras de integridade para garantir consistência forte dos dados.

Do ponto de vista de QA, o foco está em:
- validação de regras de negócio
- integridade referencial (PK/FK)
- consistência transacional
- rollback em caso de falha

Exemplos de bancos:
- PostgreSQL
- MySQL
- SQL Server
