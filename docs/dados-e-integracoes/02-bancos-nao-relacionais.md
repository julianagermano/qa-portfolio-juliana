# Bancos de Dados Não Relacionais — Visão de Qualidade

Bancos não relacionais priorizam flexibilidade, performance e escala,
com menor rigidez estrutural.

Do ponto de vista de QA, o foco está em:
- estrutura lógica dos documentos
- consistência eventual
- idempotência
- ausência de dados órfãos

Exemplos:
- MongoDB
- DynamoDB
- Redis
