# Qualidade como Plataforma (QA Platform)

Esta documentação define a Qualidade como um produto interno,
fornecendo padrões, ferramentas, gates e playbooks para apoiar
times de Produto e Engenharia.

Objetivos:
- escalar qualidade sem escalar pessoas
- reduzir decisões ad-hoc
- aumentar confiabilidade do CI/CD
- apoiar decisões de liberação
