# 📘 Projeto 2 — Processo de Qualidade de Software

Este projeto demonstra a aplicação de **processos modernos e profissionais de QA**, utilizados para garantir qualidade, rastreabilidade e clareza no desenvolvimento de software.

O foco é o fluxo de autenticação da API **Auth**, com documentação completa que pode ser utilizada em qualquer projeto real.
